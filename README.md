# Max: AI Assistant Frontend

## Tech Stack  
- JavaScript  
- React  

## Description  
Max is the frontend interface for an AI assistant that interacts with a backend service built using n8n. The application uses ngrok for secure tunnel access to the backend.

## Features  
- Header authentication for secure communication  
- Integration with various tools including GitHub, Calendar, and task management  

This README provides an overview of the tech stack and the functionalities offered by the Max AI assistant.
